# Dental Health Rounds App

ระบบรับข้อมูลนักเรียนแบบส่งลิงก์ แยกเป็นรอบการกรอก

## รันระบบ
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

เปิด http://127.0.0.1:5000

## บัญชีเริ่มต้น
- admin / admin123
- user / user123

## หมวดข้อมูล
1. ข้อมูลส่วนตัว: เลขบัตร ชื่อ วันเกิด เพศ ชั้น ห้อง ที่อยู่
2. โภชนาการ/ภาวะการเจริญเติบโต: น้ำหนัก ส่วนสูง รอบเอว สะโพก นน.อายุ สส.อายุ นน.สส.
3. เฝ้าระวังทันตสุขภาพ: ฟันผุ เหงือกอักเสบ เร่งด่วน
